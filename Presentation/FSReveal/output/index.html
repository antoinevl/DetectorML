  • Accuracy = ( TP + TN ) / ( TP + FP + TN + FN)
  • Precision = TP / ( TP + FP )
  • Recall = TP / ( TP + FN )
  • F2 = 5 Precision*Recall /(4 Precision + Recall)
  • NPP = TN / ( TN + FN )
  • Specificity = TN / ( TN + FP )
